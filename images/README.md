# Map images for scrollytelling

Export from Adobe Illustrator **using the same artboard size and format** for every export so the map doesn’t jump when scrolling.

| Filename | Step | Content |
|----------|------|--------|
| `map-intro-context.png` | Intro | New Haven broad view: Yale campus, out to Edgewood, down to Crown St, up toward Dixwell |
| `map-garages.png` | Intro | Same area with **garages and streets** emphasized |
| `map-crown-garage.png` | Beat 1 | **Crown Street Garage** highlighted |
| `map-crown-garage-price.png` | Beat 1 | Same + **$$$** (or price) annotation for “expensive rookie mistake” |
| `map-driveway.png` | Beat 2 | Friend’s **off-campus driveway** highlighted |
| `map-driveway-minivan.png` | Beat 2 | Same + **space-blocking minivan** annotation |

Use **File → Export → Export for Screens** (or Export As) and **Use Artboards** so size is consistent.
